# HORROR MOVIE DATA ANALYSIS  
  
As a horror movie fan, I have always been curious to learn more about the genre. This data provided insights on the genre more broadly than typical googling can do (and much more quickly!). While I can read think pieces on the genre and different movies within the genre all day, having quantative insights adds to the credibility of theories about the genre.  

Many of the insights can be seen in the plots in the Exploratory Visuals Folder.

## Project Structure  

#### first look files:  
1. **firstlook_cleaning.ipynb**: Cleaning and wrangling the original dataset from kaggle to more easily use for my analysis. 
2. **CLEANED_CODE.ipynb**: Cleaned block of code for accessing the data throughout my analysis.  
3. **firstlook_visuals.ipynb**: Code for visuals found in firstlook_visuals folder, these visuals helped guide questions to explore in later analysis. (stepping stone visuals) 
4. **personal_stake.ipynb**: A look at some of my horror movies! In the process, this file led to a closer look at how the dataset was compiled. *still in progress* 

#### exploratory files:  
5. **initial_questions_to_explore.md**: Notes and Questions I'd further like to explore with this data.  
6. **exploratory_visuals.ipynb**: Code for visuals found in exploratory_visuals folder, these visuals are more condensed and concise than the firstlook_visuals.
